## Local 실행 방법
local 실행 시 .env 파일에 APIKEY 추가가 필요합니다.  
[APIKEY 발급](https://www.omdbapi.com/apikey.aspx)
```
// 1. TERMINAL install
npm i -D @types/node-fetch
npm i node-fetch@2

// 2. /.env 파일에 APIKEY 추가
APIKEY=YOURKEY
```
